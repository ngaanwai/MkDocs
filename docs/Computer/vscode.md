## Mermaid 
JavaScript based diagramming and charting tool that renders Markdown-inspired text definitions to create and modify diagrams dynamically.
https://mermaid.js.org/

### Mindmaps
An example of a mindmap.
```mermaid
mindmap
    Root
        A
            B
            C
```
